# Project Style Notes

This reference is a compact reminder of the most repeated visual decisions in the repository.

## Repeated Tokens

- primary action blue: `#2563eb`
- primary hover blue: `#1d4ed8`
- deep navy: `#0f2a46`
- dark menu gradient: `#0a2a4a` to `#174d78`
- pale backgrounds: `#f8fbff`, `#edf3f9`, `#eef2f8`, `#e3e9f1`
- stroke colors: `#d2deec`, `#c8d7e9`, `#e5e7eb`

## Typography Split

- `Manrope`: public, polished, branded surfaces
- `Inter`: account, settings, dense forms, operational UI

## Radius Scale

- `10px`: controls
- `12px`: small cards and pills
- `16px` to `18px`: standard content cards
- `20px` to `24px`: feature shells and hero blocks

## Key References

- `assets/css/menu.css`
- `assets/css/index.css`
- `assets/css/auth.css`
- `assets/css/cemetery-detail.css`
- `assets/css/grave.css`
- `assets/css/profile.css`

## Fast Decision Rule

If you are adding UI and do not know what to copy:

- copy `index.css` for public discovery pages
- copy `auth.css` for polished forms
- copy `profile.css` for utility/account screens
- copy `cemetery-detail.css` for rich information detail pages
